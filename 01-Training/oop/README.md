# Training Area: Object-Oriented Programming (OOP)

Up to now, you have written **instructions**.

From this point forward, you will model **things**.

Java is not about writing steps.
It is about designing objects that:
- Hold data
- Expose behavior
- Interact with each other

If you misunderstand OOP, later phases will feel impossible.
Take this seriously.
