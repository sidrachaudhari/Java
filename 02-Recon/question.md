# Recon Questions

Answer these after fully reading the codebase.

1. Where does the program start?
2. Which class is responsible for:
   - Creating users?
   - Storing users?
   - Validating age?
3. How does data move from `Main` to storage?
4. Why is `DataStore` static?
5. What would break if `DataStore` was removed?
6. Where would you add logging if required?
7. Identify **two design issues** (not syntax issues).
8. Identify **one thing that is done well**.

Do not rush these.
The Heist depends on your answers.
